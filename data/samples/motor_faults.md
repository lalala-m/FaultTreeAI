# 电机故障知识库

## 1. 电机无法启动

### 故障现象
- 按下启动按钮，电机不转
- 电机发出嗡嗡声但不转动
- 电机保护器跳闸

### 可能原因（按概率排序）
1. 电源故障（缺相、电压过低）
2. 电机绕组短路或开路
3. 轴承损坏导致转子卡死
4. 负载过大（机械卡阻）
5. 启动电容失效（单相电机）
6. 控制电路故障（接触器、按钮）

### 故障树结构
```
顶事件：电机无法启动
├── OR
│   ├── 电源故障
│   │   ├── AND
│   │   │   ├── 缺相
│   │   │   └── 接触器故障
│   │   └── 电压过低
│   ├── 电机本体故障
│   │   ├── OR
│   │   │   ├── 绕组短路
│   │   │   └── 绕组开路
│   │   └── 轴承损坏
│   ├── 负载过大
│   │   ├── AND
│   │   │   ├── 机械卡阻
│   │   │   └── 润滑不良
│   │   └── 负载过载
│   └── 控制电路故障
│       ├── OR
│       │   ├── 启动按钮故障
│       │   └── 停止按钮故障
│       └── 接触器故障
```

### 维修建议
1. 检查电源电压和接线
2. 测量电机绕组电阻
3. 检查轴承转动是否灵活
4. 检查负载情况
5. 检查控制电路

## 2. 电机运行异常（振动/噪音）

### 故障现象
- 电机振动过大
- 异常噪音（嗡嗡声、啸叫声）
- 轴承部位温度过高

### 可能原因
1. 轴承磨损或损坏
2. 转子不平衡
3. 基础松动
4. 绕组松动
5. 冷却风扇故障

### 故障树结构
```
顶事件：电机运行异常
├── OR
│   ├── 轴承故障
│   │   ├── OR
│   │   │   ├── 轴承磨损
│   │   │   ├── 轴承损坏
│   │   │   └── 润滑不良
│   │   └── 轴承温度过高
│   ├── 机械问题
│   │   ├── OR
│   │   │   ├── 转子不平衡
│   │   │   └── 基础松动
│   │   └── 对中不良
│   └── 电气问题
│       ├── OR
│       │   ├── 绕组松动
│       │   └── 冷却风扇故障
│       └── 电源谐波
```

## 3. 电机过热

### 故障现象
- 电机表面温度过高
- 保护器报警
- 绝缘老化加速

### 可能原因
1. 负载过大（长时间过载）
2. 通风散热不良
3. 电源电压异常
4. 电机内部故障（绕组短路）
5. 环境温度过高

### 故障树结构
```
顶事件：电机过热
├── OR
│   ├── 过载运行
│   │   ├── AND
│   │   │   ├── 负载过大
│   │   │   └── 运行时间过长
│   │   └── 频繁启动
│   ├── 散热不良
│   │   ├── OR
│   │   │   ├── 冷却风扇故障
│   │   │   └── 散热片堵塞
│   │   └── 环境温度过高
│   ├── 电源问题
│   │   ├── OR
│   │   │   ├── 电压过高
│   │   │   └── 电压过低
│   │   └── 缺相运行
│   └── 电机内部故障
│       ├── OR
│       │   ├── 绕组局部短路
│       │   └── 绝缘老化
│       └── 匝间短路
```

## 4. 电机转速异常

### 故障现象
- 转速低于额定值
- 转速不稳定
- 转速过高

### 可能原因
1. 电源频率异常
2. 负载过大
3. 变频器参数设置错误
4. 电机绕组故障
5. 转子断条

### 故障树结构
```
顶事件：电机转速异常
├── OR
│   ├── 转速过低
│   │   ├── OR
│   │   │   ├── 电源电压低
│   │   │   └── 电源频率低
│   │   ├── 负载过大
│   │   └── 变频器故障
│   ├── 转速不稳定
│   │   ├── OR
│   │   │   ├── 负载波动
│   │   │   └── 电源不稳
│   │   └── 变频器参数不当
│   └── 转速过高
│       ├── 电源频率过高
│       └── 变频器参数错误
```

## 5. 电机漏电

### 故障现象
- 电机外壳带电
- 接地保护器动作
- 绝缘电阻低

### 可能原因
1. 绕组绝缘损坏
2. 接线盒进水
3. 电缆破损
4. 环境潮湿

### 故障树结构
```
顶事件：电机漏电
├── OR
│   ├── 绕组绝缘故障
│   │   ├── OR
│   │   │   ├── 绕组老化
│   │   │   ├── 绕组烧毁
│   │   │   └── 绕组受潮
│   │   └── 匝间短路
│   ├── 外部损伤
│   │   ├── OR
│   │   │   ├── 电缆破损
│   │   │   └── 接线盒进水
│   │   └── 机械损伤
│   └── 环境因素
│       └── 潮湿环境
```
