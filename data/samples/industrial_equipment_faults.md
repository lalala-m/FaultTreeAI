# 工业设备故障知识库

> 本文档包含常见工业设备的故障现象、原因分析及解决方案
> 来源：行业标准和技术手册摘要

---

## 一、电动机故障

### 1.1 电动机无法启动

**故障现象**：接通电源后，电动机不转动或转动缓慢

**可能原因**：
1. 电源问题：缺相、电压过低、开关未闭合
2. 绕组故障：绕组短路、断路、接地
3. 机械故障：轴承损坏、转子卡死、负载过大
4. 保护器故障：热继电器未复位、电机保护器跳闸

**解决方案**：
- 检查电源三相电压是否正常
- 检测绕组绝缘电阻
- 检查轴承和机械负载
- 复位保护器

### 1.2 电动机过热

**故障现象**：电动机外壳温度超过额定值

**可能原因**：
1. 过载：负载超过额定功率
2. 散热不良：风扇损坏或散热通道堵塞
3. 电压异常：电压过高或过低
4. 绕组问题：局部短路、绝缘老化

**解决方案**：
- 降低负载或更换更大功率电机
- 清理散热通道，检查风扇
- 调整电压至额定范围
- 检修或更换绕组

### 1.3 电动机振动过大

**故障现象**：运行时振动异常，伴有异常声响

**可能原因**：
1. 机械不平衡：转子不平衡、联轴器不对中
2. 轴承故障：磨损、损坏、润滑不良
3. 电磁问题：气隙不均匀、绕组不对称

**解决方案**：
- 平衡转子，校准联轴器
- 更换轴承，重新加注润滑脂
- 检查并调整气隙

### 1.4 电动机异响

**故障现象**：运行时发出异常声响

**可能原因**：
1. 轴承故障：滚动体磨损、保持架损坏
2. 气隙问题：定转子摩擦
3. 螺栓松动：机座或端盖螺栓松动

**解决方案**：
- 检查更换轴承
- 检查并调整气隙
- 紧固所有螺栓

## 二、液压系统故障

### 2.1 液压系统无压力

**故障现象**：系统无法建立工作压力，执行机构不动

**可能原因**：
1. 泵故障：内磨损、电机不转、过滤器堵塞
2. 油箱问题：油量不足、油液污染
3. 阀组故障：溢流阀设定不当、阀芯卡滞
4. 泄漏：管道泄漏、密封件损坏

**解决方案**：
- 检查泵运行状态和油液量
- 清洗或更换过滤器
- 调整溢流阀压力
- 检查并更换密封件

### 2.2 液压系统压力不稳定

**故障现象**：压力波动大，执行机构动作不平稳

**可能原因**：
1. 泵问题：变量机构故障、吸入空气
2. 阀故障：阀芯磨损、弹簧疲劳
3. 油液问题：空气混入、油温过高

**解决方案**：
- 检查泵的变量机构
- 排除系统空气
- 更换或修复阀门

### 2.3 液压系统油温过高

**故障现象**：油温超过正常工作范围（60-80°C）

**可能原因**：
1. 冷却不足：冷却器堵塞、水温高
2. 泄漏：内泄漏导致效率降低
3. 压力过高：系统压力调定过高

**解决方案**：
- 清洗冷却器，检查冷却水
- 检查泄漏并修复
- 调整系统压力

### 2.4 液压执行机构动作缓慢

**故障现象**：执行机构运动速度低于额定值

**可能原因**：
1. 流量不足：泵流量小、阀门开度小
2. 压力不足：负载过大、泄漏
3. 油液粘度过高：油温低、油品不对

**解决方案**：
- 检查泵流量和阀门开度
- 检查泄漏并修复
- 加热油液或更换合适粘度油品

## 三、PLC控制系统故障

### 3.1 PLC无法启动

**故障现象**：PLC上电后RUN灯不亮或闪烁

**可能原因**：
1. 电源故障：电源模块损坏、供电不稳
2. 程序丢失：电池电量耗尽、干扰导致程序丢失
3. CPU故障：处理器损坏

**解决方案**：
- 检查电源电压和模块状态
- 更换电池并重新下载程序
- 更换CPU模块

### 3.2 PLC通讯故障

**故障现象**：PLC与上位机或其他设备无法通讯

**可能原因**：
1. 通讯参数错误：波特率、数据位、校验位不匹配
2. 通讯模块故障：模块损坏
3. 物理连接问题：网线/RS485线缆损坏
4. 地址冲突：设备地址重复

**解决方案**：
- 检查并统一通讯参数
- 更换通讯模块
- 检查并更换通讯线缆
- 重新设定设备地址

### 3.3 输入点无信号

**故障现象**：输入点指示灯不亮，程序检测不到输入

**可能原因**：
1. 传感器故障：接近开关、按钮损坏
2. 接线问题：线缆断开、接触不良
3. 模块故障：输入模块损坏

**解决方案**：
- 检测传感器输出信号
- 检查接线是否牢固
- 更换输入模块

### 3.4 输出点无动作

**故障现象**：程序已驱动输出但执行器不动作

**可能原因**：
1. 输出模块故障：晶体管/继电器损坏
2. 接线问题：线缆断开
3. 执行器故障：电磁阀、接触器损坏
4. 负载过大：超过模块输出容量

**解决方案**：
- 检测输出模块信号
- 检查执行器电源
- 更换损坏的输出模块或执行器

## 四、离心泵故障

### 4.1 泵无法启动

**故障现象**：启动时电机不转或立即跳闸

**可能原因**：
1. 电源故障：缺相、电压过低
2. 电机故障：绕组短路、断路
3. 机械卡死：轴承损坏、叶轮卡住
4. 保护器跳闸：过载保护动作

**解决方案**：
- 检查电源和电机状态
- 检查机械部分是否卡死
- 复位保护器

### 4.2 泵流量不足

**故障现象**：出口流量低于额定值

**可能原因**：
1. 汽蚀：入口压力过低
2. 堵塞：叶轮或管道堵塞
3. 泄漏：密封件损坏、管道泄漏
4. 转速不足：电机转速低于额定

**解决方案**：
- 提高入口压力
- 清理叶轮和管道
- 更换密封件
- 检查电机转速

### 4.3 泵振动过大

**故障现象**：运行时异常振动

**可能原因**：
1. 汽蚀：工作点偏离
2. 轴承故障：磨损、损坏
3. 不平衡：叶轮磨损不均匀
4. 对中不良：联轴器对中误差大

**解决方案**：
- 调整工作点
- 更换轴承
- 平衡叶轮
- 重新对中

### 4.4 密封泄漏

**故障现象**：机械密封处泄漏

**可能原因**：
1. 密封磨损：动环、静环磨损
2. 弹簧疲劳：密封弹簧失效
3. 介质问题：腐蚀性介质、温度过高

**解决方案**：
- 更换密封件
- 选用耐腐蚀材质
- 降低介质温度

## 五、变频器故障

### 5.1 变频器过流

**故障现象**：运行中报OC（过流）故障

**可能原因**：
1. 负载过大：超过变频器容量
2. 加速时间过短：电机加速过快
3. 电机故障：绕组短路、接地
4. 输出短路：电缆损坏

**解决方案**：
- 降低负载或增大变频器容量
- 增加加速时间
- 检查电机绝缘
- 检查输出电缆

### 5.2 变频器过压

**故障现象**：运行中报OV（过压）故障

**可能原因**：
1. 电源电压过高
2. 减速时间过短
3. 负载惯性过大
4. 制动单元故障

**解决方案**：
- 检查输入电压
- 增加减速时间
- 使用制动电阻
- 检查制动单元

### 5.3 变频器过热

**故障现象**：变频器报OH（过热）故障

**可能原因**：
1. 环境温度过高
2. 散热风扇故障
3. 散热片堵塞
4. 负载过大

**解决方案**：
- 改善通风环境
- 更换散热风扇
- 清理散热片
- 降低负载

## 六、故障诊断基本方法

### 6.1 问

- 故障发生前后的情况
- 故障发生的频率
- 是否有过维修或改造

### 6.2 看

- 外观检查：是否有泄漏、损坏、变色
- 指示灯状态：运行、报警指示
- 仪表读数：压力、温度、电流

### 6.3 听

- 异常声响：撞击、摩擦、啸叫
- 运行声音是否平稳

### 6.4 触

- 振动是否正常
- 温度是否异常
- 紧固件是否松动

### 6.5 测

- 电压、电流测量
- 压力、流量测量
- 绝缘电阻测量
