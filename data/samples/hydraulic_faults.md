# 液压系统故障知识库

## 1. 液压系统无压力

### 故障现象
- 系统启动后压力表显示无压力
- 执行机构不动作
- 液压泵声音异常

### 可能原因
1. 液压泵故障（泵内磨损、轴断裂）
2. 油箱液位过低
3. 吸油管路堵塞或漏气
4. 溢流阀故障（阀芯卡住、弹簧断裂）
5. 换向阀故障（阀芯未复位）
6. 油液粘度不当

### 故障树结构
```
顶事件：液压系统无压力
├── OR
│   ├── 泵故障
│   │   ├── OR
│   │   │   ├── 泵内磨损
│   │   │   ├── 泵轴断裂
│   │   │   └── 泵反转
│   │   └── 泵电机故障
│   ├── 油箱问题
│   │   ├── OR
│   │   │   ├── 油位过低
│   │   │   └── 油液污染
│   │   └── 油液粘度不当
│   ├── 吸油管路问题
│   │   ├── OR
│   │   │   ├── 吸油管堵塞
│   │   │   ├── 吸油管漏气
│   │   │   └── 过滤器堵塞
│   │   └── 空气进入系统
│   └── 控制阀故障
│       ├── OR
│       │   ├── 溢流阀故障
│       │   ├── 换向阀故障
│       │   └── 顺序阀故障
│       └── 阀芯卡滞
```

### 维修建议
1. 检查油箱液位
2. 检查泵运转声音
3. 检查溢流阀设定压力
4. 检查吸油管路是否漏气

## 7. 液压系统泄漏
