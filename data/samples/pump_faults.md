# 泵类设备故障知识库

## 1. 离心泵无法启动

### 故障现象
- 启动时电机不转
- 泵体振动异常
- 有嗡嗡声但不运转

### 可能原因
1. 电源故障（缺相、电压低）
2. 电机故障
3. 轴承卡死
4. 叶轮卡住
5. 密封件粘连
6. 出口阀门关闭

### 故障树结构
```
顶事件：离心泵无法启动
├── OR
│   ├── 电源问题
│   │   ├── OR
│   │   │   ├── 缺相
│   │   │   ├── 电压过低
│   │   │   └── 电源开关未合上
│   │   └── 保险丝熔断
│   ├── 电机故障
│   │   ├── OR
│   │   │   ├── 电机绕组烧毁
│   │   │   └── 电机轴承损坏
│   │   └── 电机过载保护跳闸
│   ├── 机械卡阻
│   │   ├── OR
│   │   │   ├── 轴承损坏
│   │   │   ├── 叶轮卡住
│   │   │   └── 密封件粘连
│   │   └── 异物堵塞
│   └── 操作问题
│       └── 出口阀门关闭
```

## 2. 离心泵流量不足

### 故障现象
- 实际流量低于额定值
- 出口压力低
- 工艺参数无法满足

### 可能原因
1. 泵转速不够
2. 叶轮磨损
3. 吸入管路漏气
4. 过滤器堵塞
5. 汽蚀现象
6. 出口阀门开度不足

### 故障树结构
```
顶事件：离心泵流量不足
├── OR
│   ├── 泵本身问题
│   │   ├── OR
│   │   │   ├── 叶轮磨损
│   │   │   ├── 叶轮损坏
│   │   │   └── 泵转速不够
│   │   └── 泵效率下降
│   ├── 吸入问题
│   │   ├── OR
│   │   │   ├── 吸入管漏气
│   │   │   ├── 过滤器堵塞
│   │   │   └── 吸入口堵塞
│   │   └── 汽蚀
│   ├── 出口问题
│   │   ├── 阀门开度小
│   │   └── 出口压力过高
│   └── 液体问题
│       └── 液体粘度不符
```

## 3. 离心泵过载

### 故障现象
- 电机电流超过额定值
- 电机发热严重
- 保护器动作

### 可能原因
1. 流量过大（扬程不足）
2. 泵轴与电机轴不同心
3. 轴承损坏
4. 填料压得过紧
5. 液体粘度高于设计值

### 故障树结构
```
顶事件：离心泵过载
├── OR
│   ├── 负载过大
│   │   ├── AND
│   │   │   ├── 流量过大
│   │   │   └── 扬程不足
│   │   └── 出口压力过高
│   ├── 机械问题
│   │   ├── OR
│   │   │   ├── 轴承损坏
│   │   │   ├── 轴弯曲
│   │   │   └── 同心度偏差
│   │   └── 填料过紧
│   └── 液体问题
│       └── 液体粘度过高
```

## 4. 离心泵振动过大

2. 叶轮不平衡
3. 泵轴弯曲
4. 基础松动
5. 气蚀
6. 汽蚀现象
7. 管道应力

│   │   └── 轴承温度过高
│   ├── 机械不平衡
│   │   ├── OR
│   │   │   ├── 叶轮不平衡
│   │   │   ├── 泵轴弯曲
│   │   │   └── 轴套磨损
│   │   └── 转子动平衡不良
│   ├── 基础问题
│   │   ├── 基础松动
│   │   └── 减震垫损坏
│   └── 外部因素

## 6. 离心泵密封泄漏

3. O型圈损坏
4. 弹簧失效
5. 泵轴磨损
6. 安装不当

│   │   │   └── 弹簧失效
│   │   └── 密封面有杂质
│   ├── 密封件损坏
│   │   ├── OR
│   │   │   ├── O型圈老化
│   │   │   ├── O型圈损坏
│   │   │   └── 垫片损坏
│   │   └── 安装不当
│   ├── 轴问题
│   │   ├── 轴磨损
│   │   └── 轴表面划伤
