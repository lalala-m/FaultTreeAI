# PLC控制系统故障知识库

## 1. PLC无法启动/死机

### 故障现象
- PLC上电后RUN灯不亮
- PLC状态指示异常
- 程序无法运行
- 频繁重启

### 可能原因
1. 电源故障（电源模块损坏、电压不稳）
2. CPU故障（CPU损坏、过热）
3. 存储器故障（程序丢失、内存损坏）
4. 固件损坏
5. 干扰信号（电磁干扰、静电）
6. 扩展模块故障

### 故障树结构
```
顶事件：PLC无法启动
├── OR
│   ├── 电源问题
│   │   ├── OR
│   │   │   ├── 电源模块损坏
│   │   │   ├── 输入电压异常
│   │   │   └── 电源接线松动
│   │   └── 电源过载
│   ├── CPU故障
│   │   ├── OR
│   │   │   ├── CPU过热
│   │   │   ├── CPU损坏
│   │   │   └── CPU接触不良
│   │   └── 固件损坏
│   ├── 存储器故障
│   │   ├── OR
│   │   │   ├── 程序丢失
│   │   │   ├── 内存损坏
│   │   │   └── 电池电量不足
│   │   └── 存储卡故障
│   └── 干扰问题
│       ├── OR
│       │   ├── 电磁干扰
│       │   ├── 静电放电
│       │   └── 信号线干扰
│       └── 接地不良
```

## 2. PLC通讯故障

### 故障现象
